What's are the diffrences between web 1, web 2 and web 3

In Web 1 only present content of pages as the page' owner wrote it.
very simple version.
In Web 2 people could responed and write things into pages , even if they were not the owners of the page .
They used forums, chats, wikipdia comments ,Facebook.
The users begin to be content suppliers beside the initial content of page.
In Web 3 supplied needs of specific user.
Rank of pages could be diffrent between two diffrent users.
Web use multiply resources like Whattup, Facebook ,Google account .
The web build Avatar for user and save information about him.

Diffrences based query "Drugs side effects"

web 1.0: list of static websites with explanations.
same result no matter whice user search the query.
web 2.0: list of sites that realsay on users posts for drug side effects , the user see other posts like his which indexed and organized in unified medical DB
web 3.0 : list of sites so the sites with highest rank related to drugs that user use or search before.
